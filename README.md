# GTA NoSave Go

Credits to the [original .ahk script on reddit](https://www.reddit.com/r/gtaglitches/comments/okz5lg/exploit_pc_v1_nosavingsaveblock_method_ahk_replay/)

## Usage

1. Launch the .exe
2. Before going into Cayo Perico, press CTRL+F9 to toggle the firewall on.
3. After doing the heist and when you're in freeroam, go into story mode.
4. Wait until everything is loaded and make sure there are no loading icons.
5. Press CTRL+F12 to toggle the firewall off.
6. Go back into multiplayer, wait until you're loaded, press ALT+F4, and it should save, and allowing you to replay heists.
