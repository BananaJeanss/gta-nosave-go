# GTA NoSave Go

Credits to the [original .ahk script on reddit](https://www.reddit.com/r/gtaglitches/comments/okz5lg/exploit_pc_v1_nosavingsaveblock_method_ahk_replay/)

## How this works

All it does is just toggle a block to Rockstar's servers, basically preventing the heist from being saved as completed,
while still keeping the money you got from the heist.

## Usage

1. Build via either `task build` or run the taskfile.yml commands directly.
2. Launch the .exe
3. Before going into Cayo Perico, press CTRL+F9 to toggle the firewall on.
4. After doing the heist and when you're in freeroam, go into story mode.
5. Wait until everything is loaded and make sure there are no loading icons.
6. Press CTRL+F12 to toggle the firewall off.
7. Go back into GTA online, wait until you're loaded, press ALT+F4, and it should save, and allowing you to replay heists.
